# Texts

Texts are used to show information by words and/or letters.

## Titles

Titles are used to show important texts on a page.

\* Default values

| Props | Values                                                                                                                                | Description             |
| ----- | ------------------------------------------------------------------------------------------------------------------------------------- | ----------------------- |
| size  | <code>subheading</code> \| <code>sm</code> \| <code>md\*</code> \| <code>section</code> \| <code>standard</code> \| <code>huge</code> | Defines the title size. |
